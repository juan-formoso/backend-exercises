# What is NoSQL

**NoSQL (Not SQL) is a mon-relational database**

> NoSQL doesn't use tables with rows or columns
> It stores relationship data in documents

**Types of NoSQL Databases**

- Document Database (MongoDB)
- Key-value Database
- Wide-column Database
- Graph Database
